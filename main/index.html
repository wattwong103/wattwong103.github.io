Projects, coding repository, and blogs

I, as an independent researcher, have been working on various projects that aim to improve the accessibility and capacity of public transportation in Bangkok and in Thailand.

Portal: On-demand bus riding application

Portal's algorithm is designed using Python 3, relying on machine learning principles with a trial and error approach on the simulation model. We hope that we will be able to extend previous research concepts onto the model and aids in planning transportation, and logistics within zones for improved efficiency. The trial design of the application is shown on the right as this does not represent the final design.

NaviCity: Transportation routing application that helps users estimate fare and time

NaviCity is designed to help transportation users especially public transportation users that want to optimize between cost and time. NaviCity use graph theory to map out the transportation network in Bangkok and use algorithm such as A* (A-star) to optimize and find the shortest/ fastest route which users can set themselves. The trial design of the application is shown on the right as this does not represent the final design.
